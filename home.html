Follow the steps one by one

Chrome Monitor

Steps to follow:

  • Download the .crx file from the given link
  • Open chrome browser and type chrome://extensions in the url
  • Drag the downloaded .crx file to the extensions page and drop it.
  • If Extension is not enabled, then click on enable checkbox. You can see eye icon in the browser corner after the url.
  • Login into the chrome browser. Then you are good to go.
  • Don't forget to check the extension(whether it is enabled or disabled) while starting the course.(May be previously you disabled it and forgot to enable again.)
